### String Interpolation
문자열 literal에 상수, 변수를 섞어 새로운 문자열을 만드는 방법
``` swift
let multiplier = 3
let message = "\(multiplier) times 2.5 is \(Double(multiplier) * 2.5)"
// message is "3 times 2.5 is 7.5“
```
\(상수 or 변수)의 형태로 문자열 안에 넣는다.
